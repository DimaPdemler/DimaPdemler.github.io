# Tools Directory

This directory contains standalone HTML visualization tools and utilities that are accessible via direct URL but do not appear in the site navigation.

## Available Tools

- **convex_function_visualizer.html** - Interactive visualization demonstrating convex function properties
  - URL: https://www.dimademler.com/tools/convex_function_visualizer.html

## Adding New Tools

To add a new tool:
1. Place the HTML file in this directory
2. Access it at: `https://www.dimademler.com/tools/your-file.html`
3. Files here won't appear in site navigation unless explicitly linked
